Absolute Units

px

Pixel is the most common unit in CSS. It is a fixed-size unit that represents one pixel on the screen.

cm

Centimeter is an absolute unit in CSS. It is a fixed-size unit that represents one centimeter on the screen.

mm

Millimeter is an absolute unit in CSS. It is a fixed-size unit that represents one millimeter on the screen.

in

Inch is an absolute unit in CSS. It is a fixed-size unit that represents one inch on the screen.

pt

Point is an absolute unit in CSS. It is a fixed-size unit that represents one point on the screen.

pc

Pica is an absolute unit in CSS. It is a fixed-size unit that represents one pica on the screen.

Relative Units

em

Em is a relative unit in CSS. It is a scalable unit that is based on the font size of the element.

ex

Ex is a relative unit in CSS. It is a scalable unit that is based on the x-height of the element's font.

ch

Ch is a relative unit in CSS. It is a scalable unit that is based on the width of the "0" (zero) character of the element's font.

fr

Fraction is a relative unit in CSS. It is a scalable unit that is based on the available space in the container.

rem

Rem is a relative unit in CSS. It is a scalable unit that is based on the font size of the root element.

vw

Viewport width is a relative unit in CSS. It is a scalable unit that is based on the width of the viewport.

vh

Viewport height is a relative unit in CSS. It is a scalable unit that is based on the height of the viewport.

vmin

Viewport minimum is a relative unit in CSS. It is a scalable unit that is based on the smaller of the viewport's width and height.

vmax

Viewport maximum is a relative unit in CSS. It is a scalable unit that is based on the larger of the viewport's width and height.

%

Percentage is a relative unit in CSS. It is a scalable unit that is based on the size of the parent element.
